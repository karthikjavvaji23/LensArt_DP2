How the prediction was made

▤ Hybrid pipeline

Each task is delegated to the model best suited for it

ResNet50 → predicts the style from the 10 ArtBench classes (supervised, fine-tuned on labelled data).
CLIP → predicts the artist by cosine similarity between the image embedding and 41 text prompts (zero-shot).
CLIP → retrieves the top-5 similar works from the cached embedding index (cosine similarity, sub-millisecond).
Wikipedia → fetches the artist biography via REST. Met Museum acts as the fallback when Wikipedia returns nothing useful.
